# 收款码占位图说明

## 如何添加收款码

1. 准备一张真实的收款码图片（微信或支付宝）
2. 将图片重命名为 `qrcode.jpg`
3. 放置在 `public/` 目录下

## 文件位置

```
/workspace/projects/public/qrcode.jpg
```

## 格式要求

- 格式：JPG 或 PNG
- 尺寸：建议 400x400 像素或更大
- 大小：建议不超过 500KB

## 示例

如果你的收款码是微信收款码：
1. 打开微信
2. 进入"收付款"
3. 点击"收款码"
4. 截图或保存收款码图片
5. 将图片重命名为 `qrcode.jpg`
6. 放置到 `public/` 目录

## 测试

添加收款码后，访问定价页面测试：
`http://localhost:5000/pricing`

点击"扫码充值"按钮，应该能看到收款码图片。
