# Excel 文件格式说明

## 文件要求

请上传 `.xlsx` 或 `.xls` 格式的 Excel 文件。

## 文件格式

Excel 文件应包含以下列：

| 列位置 | 列名称 | 说明 | 必填 |
|--------|--------|------|------|
| A列 | 股票名称 | 股票的中文名称 | 是 |
| B列 | 股票代码 | 股票代码（如：000001） | 是 |

## 示例

| 股票名称 | 股票代码 |
|---------|---------|
| 新疆天业 | 600075 |
| 平安银行 | 000001 |
| 万科A | 000002 |
| 中国平安 | 601318 |
| 贵州茅台 | 600519 |

## 分析示例：新疆天业

**核心个股识别**：✅ 是
- 稀缺性：西北地区PVC龙头企业，拥有独特的资源优势
- 垄断性：在氯碱化工领域具有较高市场份额
- 竞争力：产品价格对业绩影响大，定价能力强
- 成长性：订单确定，产能利用率高

**催化因素**：
1. PVC价格持续上涨，预计提升毛利率3-5%
2. 下游需求回暖，订单量同比增长20%
3. 新产能投产，预计新增收入5亿元

**市场地位**：
- 西北地区最大的PVC生产企业
- 市场份额约15%，行业排名前三
- 拥有完整的氯碱化工产业链

**催化评分**：9/10（极高催化）
- 核心个股 + 产品涨价 + 订单确定 + 行业景气度高

## 注意事项

1. 第一行可以是标题行，系统会从第二行开始读取数据
2. 股票名称和代码不能为空
3. 代码可以是 6 位数字（如 000001）或带后缀（如 000001.SZ）
4. 建议一次上传不超过 50 只股票

## 使用方法

1. 准备好符合上述格式的 Excel 文件
2. 打开股票智能分析工具页面
3. 点击上传区域选择文件，或直接将文件拖拽到上传区域
4. 点击"开始分析"按钮
5. 等待分析完成，查看每只股票的详细分析报告

## 分析内容

系统将为每只股票提供以下分析：

- **综合分析摘要**：300字以内的核心分析结论
- **核心个股识别**：根据稀缺性、垄断性、竞争力、成长性判断是否为核心个股
- **市场地位**：在行业中的位置和竞争力分析
- **消息催化**：最近的利好消息和潜在催化因素
- **可炒作预期**：市场可能炒作的预期和题材
- **业务信息**：主营业务、核心产品、产能、市场份额、行业地位
- **订单确定性**：订单确定性和可持续性评分（1-10分）
- **业绩贡献**：业务对业绩的潜在贡献，量化分析
- **技术壁垒**：技术护城河和竞争优势（专利、产能、成本、渠道等）
- **热点关联**：与当前热点事件的关联性
- **催化概率**：催化发生概率评分（1-10分）
- **信息来源**：所有引用的新闻链接

## 核心个股判断标准

系统会自动识别是否为核心个股：

1. **稀缺性**：在产业链中具有独特地位或稀缺资源
2. **垄断性**：在细分领域具有较高市场份额或技术壁垒
3. **竞争力**：产品价格对业绩影响大，定价能力强
4. **成长性**：订单确定，产能利用率高，业绩增长预期明确

## 催化评分标准

- **9-10分（极高催化）**：核心个股 + 产品涨价/重大订单 + 行业景气度高
- **7-8分（高催化）**：核心个股 + 业绩超预期/政策利好 + 订单确定
- **5-6分（中等催化）**：一般个股 + 概念炒作 + 政策支持
- **3-4分（低催化）**：一般个股 + 市场热点 + 不确定性
- **1-2分（无催化）**：无明确催化因素

## 特色功能

1. **多维度搜索**：系统会从6个维度自动搜索信息
   - 公司基本信息和主营业务
   - 产品价格和业绩
   - 订单和客户
   - 政策和行业
   - 热点题材
   - 竞品和供需

2. **实时进度提示**：分析过程中会显示当前正在分析的股票名称

3. **核心个股标识**：识别出的核心个股会显示金色徽章

4. **产品价格敏感度**：特别关注产品价格变动对业绩的影响（如PVC涨价）
