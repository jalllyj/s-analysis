# Excel 文件格式说明

## 文件要求

请上传 `.xlsx` 或 `.xls` 格式的 Excel 文件。

## 文件格式

Excel 文件应包含以下列：

| 列位置 | 列名称 | 说明 | 必填 |
|--------|--------|------|------|
| A列 | 股票名称 | 股票的中文名称 | 是 |
| B列 | 股票代码 | 股票代码（如：000001） | 是 |

## 示例

| 股票名称 | 股票代码 |
|---------|---------|
| 新疆天业 | 600075 |
| 平安银行 | 000001 |
| 万科A | 000002 |
| 中国平安 | 601318 |
| 贵州茅台 | 600519 |

## 分析示例：新疆天业

**核心个股识别**：✅ 是
- 稀缺性：西北地区PVC龙头企业，拥有独特的资源优势
- 垄断性：在氯碱化工领域具有较高市场份额
- 竞争力：产品价格对业绩影响大，定价能力强
- 成长性：订单确定，产能利用率高

**催化因素**：
1. PVC价格持续上涨，预计提升毛利率3-5%
2. 下游需求回暖，订单量同比增长20%
3. 新产能投产，预计新增收入5亿元

**时间线分析**：
- **1-3个月**：
  - 2025年3月：发布一季报，预计净利润增长30%
  - 2025年4月：新产能满产，月产能提升至5万吨
  - 2025年5月：下游需求旺季，订单价格有望进一步上涨

- **3-6个月**：
  - 2025年6月：年中报业绩预告，预计上半年增长40%
  - 2025年7月：二期产能投产，总产能提升至15万吨/年
  - 2025年8月：行业需求旺季，PVC价格有望维持高位

- **大周期硬逻辑**：
  - **行业景气周期**：氯碱化工行业处于上行周期，供需格局改善
  - **大宗商品周期**：原油价格持续上涨，推动PVC等化工品价格上行
  - **国际形势**：地缘政治紧张导致能源价格上涨，利好化工品价格
  - **供需结构**：环保政策下中小产能退出，龙头公司市场份额提升
  - **持续时间**：预计行业景气周期持续18-24个月

**市场地位**：
- 西北地区最大的PVC生产企业
- 市场份额约15%，行业排名前三
- 拥有完整的氯碱化工产业链

**催化评分**：9/10（极高催化）
- 核心个股 + 大周期上行（化工行业景气） + 产品涨价 + 订单确定

## 注意事项

1. 第一行可以是标题行，系统会从第二行开始读取数据
2. 股票名称和代码不能为空
3. 代码可以是 6 位数字（如 000001）或带后缀（如 000001.SZ）
4. 建议一次上传不超过 50 只股票

## 使用方法

1. 准备好符合上述格式的 Excel 文件
2. 打开股票智能分析工具页面
3. 点击上传区域选择文件，或直接将文件拖拽到上传区域
4. 点击"开始分析"按钮
5. 等待分析完成，查看每只股票的详细分析报告

## 分析内容

系统将为每只股票提供以下分析：

### 核心维度
- **综合分析摘要**：300字以内的核心分析结论
- **核心个股识别**：根据稀缺性、垄断性、竞争力、成长性判断是否为核心个股
- **市场地位**：在行业中的位置和竞争力分析

### 时间线分析（新增）
- **1-3个月可炒作预期**：短期催化事件，标注具体时间点
  - 业绩预告/一季报发布时间
  - 订单交付时间节点
  - 产品价格短期变化
  - 政策文件发布时间
  - 产能投产时间
  - 重要会议/展会时间

- **3-6个月核心催化**：中期催化事件
  - 半年报业绩预期
  - 产能爬坡/新产品放量
  - 行业需求旺季
  - 行业政策落地
  - 重大合同履行
  - 产能利用率提升

- **大周期硬逻辑**（新增）：长期硬逻辑分析
  - 行业景气周期（上行/下行/拐点）
  - 大宗商品价格周期（存储涨价、贵金属、原油、有色金属等）
  - 国际形势变化（地缘政治、贸易战、制裁等）
  - 宏观经济周期（通胀/通缩、利率周期、汇率变化）
  - 技术革命（AI、新能源、半导体、新材料等）
  - 供需结构变化（产能出清、需求爆发、替代效应等）

- **详细催化时间线**：每个催化事件的详细信息
  - 时间范围
  - 事件名称
  - 重要性评分（1-10）
  - 催化类型（短线催化/政策催化/业绩催化/行业催化）
  - 确定性评分（1-10）
  - 详细描述

### 传统分析
- **消息催化**：最近的利好消息和潜在催化因素
- **业务信息**：主营业务、核心产品、产能、市场份额、行业地位
- **订单确定性**：订单确定性和可持续性评分（1-10分）
- **业绩贡献**：业务对业绩的潜在贡献，量化分析
- **技术壁垒**：技术护城河和竞争优势（专利、产能、成本、渠道等）
- **热点关联**：与当前热点事件的关联性
- **催化概率**：催化发生概率评分（1-10分）
- **信息来源**：所有引用的新闻链接

## 核心个股判断标准

系统会自动识别是否为核心个股：

1. **稀缺性**：在产业链中具有独特地位或稀缺资源
2. **垄断性**：在细分领域具有较高市场份额或技术壁垒
3. **竞争力**：产品价格对业绩影响大，定价能力强
4. **成长性**：订单确定，产能利用率高，业绩增长预期明确

## 催化评分标准

- **9-10分（极高催化）**：核心个股 + 大周期上行 + 短中期催化密集 + 产品涨价
- **7-8分（高催化）**：核心个股 + 中期催化确定 + 业绩超预期/政策利好
- **5-6分（中等催化）**：一般个股 + 短期催化 + 概念炒作
- **3-4分（低催化）**：一般个股 + 市场热点 + 不确定性
- **1-2分（无催化）**：无明确催化因素

## 特色功能

### 1. 多维度搜索（9个维度）
- 公司基本信息和主营业务
- 产品价格和业绩
- 订单和客户
- 政策和行业
- 热点题材
- 竞品和供需
- **未来催化（1-3个月）**（新增）
- **中期催化（3-6个月）**（新增）
- **大周期逻辑**（新增）

### 2. 时间线分析（新增）
系统会按三个时间维度梳理催化事件：
- 1-3个月：短期可炒作预期，标注具体时间点
- 3-6个月：中期核心催化，评估确定性和影响
- 大周期：长期硬逻辑，分析行业周期、大宗商品、国际形势等

### 3. 大周期硬逻辑判断（新增）
系统会识别以下大周期逻辑：
- 行业景气周期：判断当前所处位置（上行/下行/拐点）
- 大宗商品价格周期：存储、贵金属、原油、有色金属等
- 国际形势变化：地缘政治、贸易战、制裁等
- 宏观经济周期：通胀、利率、汇率等
- 技术革命：AI、新能源、半导体、新材料等
- 供需结构变化：产能出清、需求爆发、替代效应等

### 4. 实时进度提示
分析过程中会显示当前正在分析的股票名称

### 5. 核心个股标识
识别出的核心个股会显示金色徽章

### 6. 产品价格敏感度
特别关注产品价格变动对业绩的影响（如PVC涨价）
